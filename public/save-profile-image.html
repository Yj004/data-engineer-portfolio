Save Your Profile Image

Follow these steps to add your profile image to the portfolio:

  1. Right-click on your image below and select "Save Image As..."
  2. Navigate to the data-engineer-portfolio/public/images/ directory
  3. Save the file as profile-image.jpg
  4. The image will automatically appear in your portfolio with the themed styling

After saving the image, you can start the development server to see your portfolio with the image:

cd data-engineer-portfolio && npm start
