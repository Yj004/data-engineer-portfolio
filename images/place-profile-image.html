Place Your Profile Image

Follow these steps to add your profile image to the portfolio:

  1. Save the image you shared as profile-image.jpg in the public/images/ directory
  2. Make sure the file is not empty (currently it's 0 bytes)
  3. After saving the image, run npm run deploy to update your portfolio

Current Profile Image Preview:

Your profile image will appear in both the Home and About sections of your portfolio.
